********
Analyses
********

This feature allows you to create, edit details, duplicate, delete, or share analyses. You can customize new workbooks with selected
data (Genes and miRNAs, Variables, Cohorts) and the following plot types:
    - Bar Chart
    - Histogram
    - Scatter Plot
    - Violin Plot
    - Cubby Hole Plot
    - SeqPeek
    - OncoPrint
    - OncoGrid
